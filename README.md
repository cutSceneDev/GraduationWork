# Student_Tester_front
front SPA with Vue.js

1. clone rep to your computer:
       => git clone https://github.com/cutSceneDev/Student_Tester_front.git
2. enter folder:
       => cd student_tester_front
3. install packages:
       => npm install
4. start server:
       => npm run dev
